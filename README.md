# PressPage Assignment (press-page-assignment)

PressPage Frontend Assignment

## Install the dependencies

```bash
npm install
```

### Start the app in development mode

```bash
npm run serve
```

### Lint the files

```bash
npm run lint
```

### Format the files

```bash
npm run format
```

### Build the app for production

```bash
npm run build
```
